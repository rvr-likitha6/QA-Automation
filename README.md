# QA-Automation

A repository containing automated testing projects using industry-standard QA frameworks.

## Frameworks Covered

### 🥒 Cucumber
Behavior-Driven Development (BDD) testing using Cucumber with Gherkin syntax.
- Writing feature files with Given/When/Then steps
- Step definition implementations
- BDD test execution and reporting

### 🎭 Playwright
End-to-end testing using Microsoft Playwright.
- Browser automation across Chromium, Firefox, and WebKit
- UI interaction and assertion tests
- Headless and headed test execution

## Getting Started

### Cucumber
```bash
cd Cucumber
npm install
npm test
```

### Playwright
```bash
cd Playwright
npm install
npx playwright test
```

## Prerequisites

- [Node.js](https://nodejs.org/) v14+
- npm or yarn

## Purpose

This repository is for learning and practicing QA automation — covering BDD with Cucumber and end-to-end browser testing with Playwright.
